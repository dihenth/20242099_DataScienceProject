# Renewable Electricity Dashboard

Streamlit dashboard for the 5DATA004C Data Science Project Lifecycle coursework.

## Dataset
- Source: World Bank
- Indicator: Electricity production from renewable sources, excluding hydroelectric (% of total)
- Indicator code: EG.ELC.RNWX.ZS
- Link: https://data.worldbank.org/indicator/EG.ELC.RNWX.ZS
